# Machine Learning Homework
HMM Speech Recognition

This project implements a simple speech recognition example using a Hidden Markov Model (HMM) and the Viterbi algorithm.

The goal is to determine the most likely phoneme sequence for the observation sequence:

High → Low

The possible hidden states (phonemes) are:

e , v

Transition probabilities and emission probabilities are used to compute the most probable sequence.

Repository structure:

code/ : Python implementation of the Viterbi algorithm
report/ : Explanation of the method and results
data/ : Observation sequence
results/ : Output produced by the program
